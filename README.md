## Installation

I use [dotter](https://github.com/SuperCuber/dotter) to manage and deploy my dotfiles.
When that is installed there are a bunch of other dependencies that need installing.

* sway

  * swaylock
  * i3status
  * pamixer
  * playerctl
  * amixer
  * light
  * foot
* foot
* zsh

  * zsh-completions
  * zsh-syntax-highlighting
  * starship
  * bat
  * exa
  * zoxide
  * dog
* kickoff
* mako
* neovim
* git
